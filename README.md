# 브랜드 아이덴티티

- 브랜드명: NatureBorn (네이처본)
- 타겟: 20~35세 환경 의식 있는 여성
- 톤앤매너: 자연친화적, 따뜻함, 청정함, 미니멀
- USP: "자연에서 온 성분 그대로, 피부에 닿는 첫 번째 자연"
- 핵심 메시지(한 문장): "자연이 만든 것을, 자연 그대로 담았습니다."

씬 1 - 오프닝 (3초)

## 씬 1 | Intro - 자연의 시작

- 씬 길이: 3초
- 목표 메시지: "청정 자연의 감성을 첫 컷에 각인"

- 화면 구성:
  · 구도: 와이드샷 → 서서히 줌인
  · 피사체: 이슬 맺힌 초록 잎사귀, 아침 햇살
  · 배경: 안개 낀 숲속 / 맑은 계곡
  · 텍스트: 없음 (순수 비주얼)

- 내레이션/카피: (없음 - 자연 소리만)

- 사용 도구:
  · 이미지: Midjourney → 키비주얼 생성
  · 비디오: Runway Gen-3 → 이미지에 모션 추가
  · 오디오: ElevenLabs 또는 Suno → 새소리+물소리

- 프롬프트:
  "dewy green leaves in misty forest, morning golden hour light, 
   macro photography, soft bokeh, clean and pure atmosphere, 
   8k, photorealistic"

- 출력 결과 요약: 청정 자연 감성의 키비주얼 확보
- 파일명: scene01_nature.png / scene01_motion.mp4

씬 2 - 제품 등장 (3초)

## 씬 2 | Product - 제품 클로즈업

- 씬 길이: 3초
- 목표 메시지: "자연 성분이 담긴 제품임을 시각적으로 전달"

- 화면 구성:
  · 구도: 제품 중앙 배치, 클로즈업
  · 피사체: 크림/세럼 용기 + 주변에 꽃잎/허브 잎
  · 배경: 흰 대리석 또는 나무 질감 테이블
  · 텍스트: 없음

- 내레이션/카피: "자연이 만든 것을,"

- 사용 도구:
  · 이미지: Adobe Firefly → 제품 광고 특화 생성
  · 비디오: Pika Labs → 부드러운 회전 모션
  · 오디오: 씬1 오디오 연속

- 프롬프트:
  "natural skincare product on white marble surface, 
   surrounded by fresh herbs and flower petals, 
   soft natural lighting, luxury minimal aesthetic, 
   commercial photography style, 8k"

- 출력 결과 요약: 자연 소재와 어우러진 제품 비주얼 확보
- 파일명: scene02_product.png / scene02_motion.mp4

씬 3 - 브랜드 메시지 (4초)

## 씬 3 | Closing - 브랜드 메시지 + CTA

- 씬 길이: 4초
- 목표 메시지: "브랜드명과 슬로건을 각인"

- 화면 구성:
  · 구도: 중앙 정렬, 미니멀
  · 피사체: 로고 + 슬로건 텍스트
  · 배경: 부드러운 크림/아이보리 그라데이션
  · 텍스트: "NatureBorn" + "자연 그대로 담았습니다"

- 내레이션/카피: "자연 그대로 담았습니다. — NatureBorn"

- 사용 도구:
  · 이미지: Canva AI → 로고/텍스트 배경 생성
  · 비디오: Runway → 페이드인 모션
  · 오디오: ElevenLabs TTS → 내레이션 음성 생성

- 프롬프트:
  "minimal clean background, soft cream and sage green gradient, 
   organic texture, nature-inspired, luxury skincare brand aesthetic"

- 출력 결과 요약: 브랜드 클로징 배경 이미지 확보
- 파일명: scene03_closing.png / scene03_motion.mp4 / narration.mp3

3단계: 이미지 생성 실전 가이드
🖼️ 추천 도구 & 사용법
Midjourney (씬 1 추천)

# Discord에서 /imagine 명령어 사용

프롬프트 예시:
/imagine dewy green leaves in misty forest, morning golden hour, 
macro photography, soft bokeh, pure atmosphere, 
--ar 16:9 --v 6 --style raw --q 2

💡 핵심 파라미터:
--ar 16:9   → 영상 비율 맞추기
--v 6       → 최신 버전 사용
--style raw → 과장 없는 사실적 표현
--q 2       → 고품질
Adobe Firefly (씬 2 추천)

# firefly.adobe.com 접속

장점: 상업적 사용 안전, 제품 광고에 특화
사용법:
1. "Text to Image" 선택
2. 프롬프트 입력
3. Content type: "Photo" 선택
4. Style: "Minimalist" 적용
🔄 스타일 일관성 유지 방법


✅ 모든 씬에 공통으로 넣을 키워드:
"soft natural lighting, clean minimal aesthetic, 
 nature-inspired, organic, fresh green tones, 
 cream and white palette"

✅ Midjourney sref(스타일 레퍼런스) 활용:
씬 1에서 마음에 드는 이미지 생성 후
→ 해당 이미지 URL을 --sref [URL]로 다음 씬에 적용
→ 화풍 통일 가능!

Runway 사용법 (씬 1)

1. runwayml.com 접속 → Gen-3 Alpha 선택
2. 생성한 이미지 업로드
3. Motion Brush로 움직임 방향 지정
   예: 잎사귀 → 살짝 흔들림 / 카메라 → 천천히 줌인
4. 프롬프트: "gentle breeze, slow zoom in, cinematic"
5. Duration: 3초 설정
6. Generate!

💡 크레딧 절약 팁:
- 먼저 4초 버전으로 테스트
- 마음에 들면 최종본 생성
- 불필요한 재생성 최소화

내레이션 생성 - ElevenLabs

# elevenlabs.io 접속

1. Text to Speech 선택
2. 텍스트 입력:
   "자연이 만든 것을, 자연 그대로 담았습니다. 네이처본."
3. Voice 선택: 
   → 따뜻하고 차분한 여성 목소리 추천
   → "Rachel" 또는 "Bella" 스타일
4. Stability: 75% / Similarity: 80% 설정
5. Generate & Download (MP3)
6단계: 편집 통합 가이드
✂️ CapCut 사용 (무료, 쉬움)

편집 순서:

1. 새 프로젝트 → 16:9 비율 설정 (1920x1080)

2. 영상 클립 배치:
   [scene01_motion.mp4] [scene02_motion.mp4] [scene03_motion.mp4]
   ←    3초    →        ←    3초    →        ←    4초    →

3. 전환 효과:
   씬1→씬2: 부드러운 페이드 (0.5초)
   씬2→씬3: 화이트 페이드 (0.5초)

4. 오디오 레이어:
   - 트랙1: 배경음악 (전체 구간, 볼륨 60%)
   - 트랙2: 자연소리 (씬1 구간, 볼륨 80%)
   - 트랙3: 내레이션 (씬3 구간, 볼륨 100%)

5. 자막 추가 (씬3):
   폰트: 깔끔한 산세리프 / 색상: 크림 화이트
   "자연 그대로 담았습니다"
   "NatureBorn"

6. 색보정:
   전체: 밝기 +5, 채도 -10, 색온도 따뜻하게
   → 자연스럽고 따뜻한 톤 통일

7. 내보내기:
   해상도: 1080p / 프레임: 30fps / 형식: MP4
