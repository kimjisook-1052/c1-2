# 브랜드 아이덴티티

- 브랜드명: NatureBorn (네이처본)
- 타겟: 20~35세 환경 의식 있는 여성
- 톤앤매너: 자연친화적, 따뜻함, 청정함, 미니멀
- USP: "자연에서 온 성분 그대로, 피부에 닿는 첫 번째 자연"
- 핵심 메시지(한 문장): "자연이 만든 것을, 자연 그대로 담았습니다."

씬 1 - 오프닝 (2초)

## 씬 1 | Intro - 자연의 시작

- 씬 길이: 2초
- 목표 메시지: "청정 자연의 감성을 첫 컷에 각인"

- 화면 구성:
  · 구도: 와이드샷 → 서서히 줌인
  · 피사체: 이슬 맺힌 초록 잎사귀, 아침 햇살
  · 배경: 안개 낀 숲속 / 맑은 계곡
  · 텍스트: 없음 (순수 비주얼)

- 내레이션/카피: (없음 - 자연 소리만)

- 사용 도구:
  · 이미지: Midjourney → 키비주얼 생성
  · 비디오: Runway Gen-3 → 이미지에 모션 추가
  · 오디오: ElevenLabs 또는 Suno → 새소리+물소리

- 프롬프트:
  "dewy green leaves in misty forest, morning golden hour light, 
   macro photography, soft bokeh, clean and pure atmosphere, 
   8k, photorealistic"

- 출력 결과 요약: 청정 자연 감성의 키비주얼 확보
- 파일명: scene01_nature.png / scene01_motion.mp4

씬 2 - 제품 등장 (2초)

## 씬 2 | Product - 제품 클로즈업

- 씬 길이: 2초
- 목표 메시지: "자연 성분이 담긴 제품임을 시각적으로 전달"

- 화면 구성:
  · 구도: 제품 중앙 배치, 클로즈업
  · 피사체: 크림/세럼 용기 + 주변에 꽃잎/허브 잎
  · 배경: 흰 대리석 또는 나무 질감 테이블
  · 텍스트: 없음

- 내레이션/카피: "자연이 만든 것을,"

- 사용 도구:
  · 이미지: Adobe Firefly → 제품 광고 특화 생성
  · 비디오: Pika Labs → 부드러운 회전 모션
  · 오디오: 씬1 오디오 연속

- 프롬프트:
  "natural skincare product on white marble surface, 
   surrounded by fresh herbs and flower petals, 
   soft natural lighting, luxury minimal aesthetic, 
   commercial photography style, 8k"

- 출력 결과 요약: 자연 소재와 어우러진 제품 비주얼 확보
- 파일명: scene02_product.png / scene02_motion.mp4

### 이 과제를 마친 후, 학습자는 아래를 스스로 설명할 수 있어야 한다.

1.텍스트/이미지/비디오/오디오 생성 도구의 역할과 차이(강점/약점)를 말로 설명할 수 있다.

1️⃣ 텍스트 (ChatGPT)
항목	내용
역할	기획, 프롬프트 작성, 내레이션 스크립트
강점	빠름, 저렴, 아이디어 생성
약점	이미지/영상/소리 못 만듦
2️⃣ 이미지 (Midjourney)
항목	내용
역할	씬별 정지 이미지 생성
강점	고품질, 빠름, 스타일 일관성 가능
약점	동영상 불가, 얼굴 디테일 어려움, 크레딧 소모
3️⃣ 비디오 (Runway)
항목	내용
역할	이미지에 모션 추가 (정지→영상)
강점	자연스러운 움직임, 세밀한 제어
약점	크레딧 많이 소모, 생성 시간 오래
4️⃣ 오디오 (Suno + ElevenLabs)
항목	내용
역할	배경음악(Suno) + 내레이션(ElevenLabs)
강점	빠름, 자연스러움, 로열티프리
약점	품질 편차, 크레딧 소모


2.기획 의도에 맞는 결과물을 얻기 위해 프롬프트를 어떻게 설계·수정했는지 설명할 수 있다.

프롬프트 설계 3단계:
1️⃣ 기획 의도 명확히 (톤, 메시지, 용도)
2️⃣ 구체적 단어 사용 (추상적 X)
3️⃣ 기술적 설정 추가 (비율, 길이, 스타일)


3. 동일한 요구를 여러 도구로 시도했을 때 결과 차이를 비교하고, 도구 선택의 이유를 설명할 수 있다.

✅ Runway 선택 이유
🎯 광고는 "품질"이 최우선

1️⃣ 자연스러운 모션
   → 시청자가 어색함을 느끼면 광고 효과 ↓
   → Runway만 영화 같은 퀄리티 제공

2️⃣ 브랜드 이미지 보호
   → 네이처본 = 신뢰감, 고급스러움
   → 어색한 영상은 브랜드 손상

3️⃣ 비용 vs 효과
   → 크레딧 소모 크지만 (100+)
   → 결과물이 우수해서 재촬영 불필요
   → 최종적으로 경제적

4️⃣ 프롬프트 제어
   → Motion Brush로 세밀한 조절 가능
   → 원하는 정확한 모션 구현


4. 생성된 미디어 소스를 해상도/비율/톤앤매너 관점에서 정리하고, 하나의 광고 영상으로 통합하는 흐름을 설명할 수 있다.

⏱️ 3초 ~ 4초 (씬2: 제품)

   배경음악 계속
   톤: 고급스럽고 신뢰감

⏱️ 5초 ~ 8초 (씬3: 클로징)

   텍스트: "자연이 만든 것을, 자연 그대로 담았습니다"
