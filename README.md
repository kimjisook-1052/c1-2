2차 미션 학습지 - AI 기반 브랜드 광고 패키지 완성

1. 무엇을 해야 하나요? (핵심 목표)
단순히 AI로 그림 한 장 그리는 게 아니라, **[기획 → 이미지 생성 → 영상으로 변환 → 소리 넣기 → 최종 편집]**이라는 전체 과정을 직접 설계해 보는 것이 목표입니다.

2. 구체적으로 어떤 결과물을 내야 하나요?
두 가지만 제출하면 됩니다.

기획서 (PDF): 어떤 브랜드를 만들 건지, 각 장면(씬)은 어떻게 구성할 건지 적은 문서입니다.
광고 영상 (MP4): AI로 만든 10초 이내의 짧은 영상입니다.

3. 제작 단계 (이 순서대로 해보세요!)
1단계: 브랜드 정하기 (기획)
가상의 브랜드나 실제 브랜드를 하나 정하세요. (예: 미래형 운동화, 친환경 카페, 우주 여행 서비스 등)
핵심 메시지를 정합니다. (예: "세상에서 가장 가벼운 신발, 에어워크")

2단계: 스토리보드 짜기 (설계도)
10초를 2~3개의 장면으로 나눕니다.
각 장면마다 어떤 화면이 나올지, 어떤 대사가 나올지, 어떤 AI 도구를 쓸지 미리 적어둡니다.
중요: 프롬프트(명령어)를 넣었을 때 한 번에 안 나올 수 있어요. "처음엔 이렇게 넣었는데 결과가 이상해서, 이렇게 수정했다"라는 기록을 꼭 남겨야 합니다.

3단계: AI로 재료 만들기 (제작)
이미지 AI: (예: Midjourney, DALL-E) 브랜드 느낌에 맞는 멋진 사진을 만듭니다.
비디오 AI: (예: Runway, Luma, Kling) 만든 이미지를 움직이게 만듭니다.
오디오 AI: (예: Suno, ElevenLabs) 배경음악이나 멋진 목소리(나레이션)를 만듭니다.

4단계: 편집하기 (완성)
캡컷(CapCut)이나 프리미어 같은 도구로 영상과 소리를 합칩니다.
마지막 3~5초에는 반드시 브랜드 로고나 이름이 나와야 합니다.

4. 꼭 지켜야 할 규칙 (주의사항)
AI만 사용하기: 직접 촬영한 영상이나 유료 스톡 영상은 쓰면 안 됩니다. 오직 AI가 만든 소스만 쓰세요.
10초 이내: 너무 길 필요 없습니다. 짧고 강렬하게 만드세요.
일관성 유지: 1번 장면은 만화 같은데, 2번 장면은 실사 사진 같으면 안 되겠죠? 전체적인 분위기(톤앤매너)를 맞추는 게 실력입니다.

# 브랜드 아이덴티티

- 브랜드명: NatureBorn (네이처본)
- 타겟: 20~35세 환경 의식 있는 여성
- 톤앤매너: 자연친화적, 따뜻함, 청정함, 미니멀
- USP: "자연에서 온 성분 그대로, 피부에 닿는 첫 번째 자연"
- 핵심 메시지(한 문장): "자연이 만든 것을, 자연 그대로 담았습니다."

스토리보드 설계

씬	길이	내용	구조

씬 1 - 오프닝 (2초)

## 씬 1 | Intro - 자연의 시작

- 씬 길이: 2초
- 목표 메시지: "청정 자연의 감성을 첫 컷에 각인"

- 화면 구성:
  · 구도: 와이드샷 → 서서히 줌인
  · 피사체: 이슬 맺힌 초록 잎사귀, 아침 햇살
  · 배경: 안개 낀 숲속 / 맑은 계곡
  · 텍스트: 없음 (순수 비주얼)

- 내레이션/카피: (없음 - 자연 소리만)

- 사용 도구:
  · 이미지: Midjourney → 키비주얼 생성
  · 비디오: Runway Gen-3 → 이미지에 모션 추가
  · 오디오: ElevenLabs 또는 Suno → 새소리+물소리

- 프롬프트:
  "dewy green leaves in misty forest, morning golden hour light, 
   macro photography, soft bokeh, clean and pure atmosphere, 
   8k, photorealistic"

- 출력 결과 요약: 청정 자연 감성의 키비주얼 확보
- 파일명: scene01_nature.png / scene01_motion.mp4

씬 2 - 제품 등장 (2초)

## 씬 2 | Product - 제품 클로즈업

- 씬 길이: 2초
- 목표 메시지: "자연 성분이 담긴 제품임을 시각적으로 전달"

- 화면 구성:
  · 구도: 제품 중앙 배치, 클로즈업
  · 피사체: 크림/세럼 용기 + 주변에 꽃잎/허브 잎
  · 배경: 흰 대리석 또는 나무 질감 테이블
  · 텍스트: 없음

- 내레이션/카피: "자연이 만든 것을,"

- 사용 도구:
  · 이미지: Adobe Firefly → 제품 광고 특화 생성
  · 비디오: Pika Labs → 부드러운 회전 모션
  · 오디오: 씬1 오디오 연속

- 프롬프트:
  "natural skincare product on white marble surface, 
   surrounded by fresh herbs and flower petals, 
   soft natural lighting, luxury minimal aesthetic, 
   commercial photography style, 8k"

- 출력 결과 요약: 자연 소재와 어우러진 제품 비주얼 확보
- 파일명: scene02_product.png / scene02_motion.mp4

## 씬 3 | Closing - 브랜드 메시지 + CTA

### 이 과제를 마친 후, 학습자는 아래를 스스로 설명할 수 있어야 한다.

1.텍스트/이미지/비디오/오디오 생성 도구의 역할과 차이(강점/약점)를 말로 설명할 수 있다.

1️⃣ 텍스트 (ChatGPT)
항목	내용
역할	기획, 프롬프트 작성, 내레이션 스크립트
강점	빠름, 저렴, 아이디어 생성
약점	이미지/영상/소리 못 만듦
2️⃣ 이미지 (Midjourney)
항목	내용
역할	씬별 정지 이미지 생성
강점	고품질, 빠름, 스타일 일관성 가능
약점	동영상 불가, 얼굴 디테일 어려움, 크레딧 소모
3️⃣ 비디오 (Runway)
항목	내용
역할	이미지에 모션 추가 (정지→영상)
강점	자연스러운 움직임, 세밀한 제어
약점	크레딧 많이 소모, 생성 시간 오래
4️⃣ 오디오 (Suno + ElevenLabs)
항목	내용
역할	배경음악(Suno) + 내레이션(ElevenLabs)
강점	빠름, 자연스러움, 로열티프리
약점	품질 편차, 크레딧 소모


2.기획 의도에 맞는 결과물을 얻기 위해 프롬프트를 어떻게 설계·수정했는지 설명할 수 있다.

프롬프트 설계 3단계:
1️⃣ 기획 의도 명확히 (톤, 메시지, 용도)
2️⃣ 구체적 단어 사용 (추상적 X)
3️⃣ 기술적 설정 추가 (비율, 길이, 스타일)


3. 동일한 요구를 여러 도구로 시도했을 때 결과 차이를 비교하고, 도구 선택의 이유를 설명할 수 있다.

비교표

✅ Runway 선택 이유
🎯 광고는 "품질"이 최우선

1️⃣ 자연스러운 모션
   → 시청자가 어색함을 느끼면 광고 효과 ↓
   → Runway만 영화 같은 퀄리티 제공

2️⃣ 브랜드 이미지 보호
   → 네이처본 = 신뢰감, 고급스러움
   → 어색한 영상은 브랜드 손상

3️⃣ 비용 vs 효과
   → 크레딧 소모 크지만 (100+)
   → 결과물이 우수해서 재촬영 불필요
   → 최종적으로 경제적

4️⃣ 프롬프트 제어
   → Motion Brush로 세밀한 조절 가능
   → 원하는 정확한 모션 구현


4. 생성된 미디어 소스를 해상도/비율/톤앤매너 관점에서 정리하고, 하나의 광고 영상으로 통합하는 흐름을 설명할 수 있다.

⏱️ 0초 ~ 2초 (씬1: 오프닝)
   자연 이미지 + 줌인 모션
   배경음악 시작
   톤: 따뜻하고 신비로움

⏱️ 3초 ~ 4초 (씬2: 제품)
   제품 이미지 + 회전 모션
   배경음악 계속
   톤: 고급스럽고 신뢰감

⏱️ 5초 ~ 8초 (씬3: 클로징)
   배경 이미지
   텍스트: "자연이 만든 것을, 자연 그대로 담았습니다"
   로고: "NATURBORN"
   내레이션: 같은 텍스트 음성
   배경음악 + 내레이션
   톤: 따뜻하고 신뢰감 있음

🎵 음향:
   배경음악 (0~08초, 60%)
   내레이션 (6~08초, 100%)
   → 자연스러운 페이드아웃
